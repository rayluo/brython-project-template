Without a doubt, you've seen fancier demos than this in Brython's gallery.


    
However, view
      
      the source code from this template's repo
      ...
    


    

    It contains no implementation to those ASCII arts!
    Instead, you will find a normal dependency declaration file "brequirements.txt",
    and an `import cowpy` code in our `main.py`.
    


    

    This is made possible by a `pip`-like tool named - what else - `brip`.
    That is to say, you no longer have to vendorize your dependency and pack your entire static website as a PyPI package for deployment.
    Check out
    the README of this project
    to see how to setup all these.
